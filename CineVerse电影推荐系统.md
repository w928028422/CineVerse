# CineVerse电影推荐系统

## Core Features

- 电影信息展示

- 多维度搜索

- 收藏观影清单

- 个性化推荐

- 响应式设计

## Design

采用Material Design深色主题，以深蓝灰色为主调配合金色强调，营造沉浸式观影氛围。响应式设计支持桌面端12列、平板端8列、移动端4列栅格布局，电影卡片采用圆角设计配合柔和阴影效果
